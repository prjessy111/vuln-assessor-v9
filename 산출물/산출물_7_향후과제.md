# 향후 과제 (로드맵)

> 본 과제는 **AI 기반 보안 진단 정합성 검증**의 핵심(수집·AI판정·정합성·CVE)을 달성했다.
> 아래는 차기 확장·고도화 과제로, 범위가 넓어 **단계적 리소스 배분**을 권고한다.
> (현재 완료: Linux/Windows 수집·AI 2-pass 진단, SecuMS 정답지 대비 정합성 84~88%(사내 LLM), 정합성 리포트, CVE, 기본 관리화면, **자율 진단 에이전트(Claude Skill) 구현·실측** — 아래 B 참조)

---

## A. 점검 대상 확장
| 과제 | 내용 | 난이도 |
|---|---|---|
| **타 OS 지원** | AIX · HP-UX · Solaris 수집 스크립트 + AI 판정 정합성 확보 (sh에 분기 일부 존재, 정합성 검증 필요) | 중 |
| **DB / WAS / 네트워크** | Oracle·MySQL·Tomcat·WebLogic + 네트워크 장비 점검 항목 확장 (OS 외 영역) | 중 |
| **CVE 보강** | NVD/MSRC 피드 동기화 주기화, 매칭 커버리지·KEV 확대 | 중 |
| **클라우드/컨테이너** | Linux 외 클라우드 인스턴스·컨테이너 점검 | 상 |

## B. 신규 항목 자동 점검 — Claude Skill Agent (✅ 달성 · 구현 완료)
> 당초 차기 과제로 분류했으나 **본 과제 내 구현·실측 완료**. 상세 — [산출물_12](산출물_12_자율진단에이전트.md).

| 과제 | 내용 | 상태 |
|---|---|---|
| **점검 절차 자동 생성** | 신규 항목을 **자연어로 정의**하면 점검 스크립트(sh/ps1)를 사내 LLM이 자동 생성 + 위험명령 안전게이트 | ✅ 완료 |
| **수집→판정 자동화** | 생성 스크립트 실행(SSH/WinRM) → raw 자동 판정 → 교차검증(3소스) → 메신저 보고. 사람 게이트 2개 | ✅ 완료 |
| **실측** | jessy207 SSH 실행→취약 판정(신뢰도 1.00)·조치 가이드 자동, 텔레그램 보고, 자동 테스트 20케이스 통과 | ✅ |
| → 차기 | 정확도 정량 벤치(재현율·정밀도·오탐율, LSAP vs Claude) | 진행 |

## C. 정합성·판정 고도화
| 과제 | 내용 | 난이도 |
|---|---|---|
| **정합성 추가 확인·경계 항목 안정화** | 정답지 대비 정합성 추가 검증 + LLM 실행별 변동(±수%) 축소(N회 다수결) | 중 |
| **조치 후 재검증 (Closed-loop)** | 조치완료 항목을 재진단해 **실제 해소 여부 자동 확인** (감사 핵심) | 중 |
| **수집 완전성 보강** | 권한 부족 항목(sudo 수집), 항목↔수집명령 매핑(crosswalk) 정기 검증 | 중 |

## D. 관리 기능 고도화 (구현 중/미완)
| 과제 | 내용 | 상태 |
|---|---|---|
| **예외/제외 고도화** | 점검항목 선택 · 호스트(IP 트리) · 계정 · **CSV/Excel 일괄 import** · 검색 · 날짜·제목·관련근거 필터 | 일부 구현 |
| **예외 생명주기** | 예외 보기(상세) · 만료 관리 · **만료 예외 재신청** | 미구현 |
| **사후 예외** | 진단 결과 화면에서 항목별 예외 | 구현 |
| **조치 관리** | 날짜·검색 필터, 조치 후 재검증 연동 | 일부 구현 |
| **서버 관리** | 헬스체크 실수집(CPU/메모리/디스크) **완료** · 서버 등록/CSV 임포트/상세/전체 헬스체크 버튼 | 로직 완료, 일부 화면 미연결 |

## 권고
- **즉시 가치**: B(신규항목 AI Agent) **✅달성** · C(조치 후 재검증) — "AI가 검증하고 닫는다"는 완결 스토리.
- **확장성**: A(타 OS) — 적용 대상 확대.
- **운영 필수(금융)**: E-감사로그 · D-예외 생명주기.
- 범위가 **5인 팀급**이므로, 핵심(B·C) → 확장(A) → 운영(D·E·F) 순 단계적 추진을 권고한다.
