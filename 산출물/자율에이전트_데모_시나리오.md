## 2. 편집 메모
- **대기 3구간**(② 생성 / ③-A 실행+판정 / ⚖ 교차검증)은 각 ~30초 → **컷 또는 4~8배속** 처리. 그래야 자막(대기 cue 6·11·12·14가 짧게 잡혀 있음)과 맞습니다.
- 폰 텔레그램 도착 장면은 **화면 위 PIP**로 넣거나 별도 컷으로.
- 녹화 후 실제 단계 등장 시각을 알려주시면 SRT 타임코드를 **정밀 싱크**해 드립니다.

## 3. 강조 포인트 (평가용)
- **자연어 한 줄 → 스크립트 자동 생성** (사람이 룰 안 짬)
- **사람 게이트 2개**(실행 승인 / 판정 확정) — 안전
- **사내 LLM 전수 + 외부 전송 차단** — 보안
- **실제 서버 SSH 실행 + 정확 판정**(취약·근거·조치)
- **Hermes 메신저 보고** — raw 미전송, 요약만 (운영 연동)
