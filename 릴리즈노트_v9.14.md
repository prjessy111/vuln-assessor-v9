## 2. CVE/SCA 대폭 확장 (이번 릴리스 핵심) ⭐
이전: OS 패치(rpm/핫픽스) CVE만 → **통합 CVE 보고서**:
- **설치 프로그램** (Windows 레지스트리 Uninstall / Linux rpm·dpkg)
- **오픈소스 라이브러리** 신규:
  - JAR (파일명 파싱, `INV-JAR`)
  - 네이티브 .dll/.so (strings 버전 추출, `INV-NATIVELIB` — OpenSSL/zlib/expat/sqlite/pcre/libxml2/poco/tinyxml/sigar)
  - SBOM (CycloneDX/SPDX, purl 정확 식별, `INV-SBOM`)
- **NVD API 2.0 연동** (`scripts/sync-nvd-api.js`) — 구 1.1 피드 폐기(403) 대응. cve-db.json 2,698건 → 매칭가능 2,564건
- **NVD CPE 어댑터** (`nvdCpe.js`) — NVD 전체를 매처에 자동 투입
- **CISA KEV 제품 교차검증** (`kevCrosscheck.js`)
- **백포트 인지 판정** (`judge.judgeSoftwareAll`) — 번들=유력 / 설치프로그램·OS패키지=검토필요(과탐 억제)
- **배포판 advisory 백포트 결정론** (`distroAdvisory.js`, RHEL/Ubuntu, `CVE_BACKPORT_CHECK=1`)
- **표기 근거 명시** — 항목별 버전범위·출처(NVD/큐레이션)·수정버전·백포트 근거
- **멀티 OS**: Windows / CentOS·RHEL / Ubuntu·Debian
- **CVE 자동 일일 갱신** (앱 내장 스케줄러, `CVE_AUTO_SYNC`)
