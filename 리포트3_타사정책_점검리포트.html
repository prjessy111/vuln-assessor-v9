구현 방법 (앱 자동 생성)

단계내용상태
1. 항목 매핑타사코드 ↔ 우리 SRV/수집키 (위 표)✅ 확정
2. 판정 연동대상 서버 진단(리포트1)의 SRV별 양호/취약/N/A → 타사 항목에 매핑룰셋 코드화 필요
3. 점수·등급항목 1~5점 가중, 준수율→A~F 등급 산출룰셋 코드화 필요
4. 출력본 양식에 자동 채워 리포트3 생성2·3 완료 후 자동

→ 매핑·양식·점수체계는 완성. 남은 건 2·3(판정연동+점수룰)을 앱 룰셋(JSON)으로 코드화하면 서버별 리포트3가 자동 생성됨.
