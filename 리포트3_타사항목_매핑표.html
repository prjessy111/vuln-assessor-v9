리포트3을 "타사 정책"으로 만드는 절차

  1. 위 매핑표를 룰셋 데이터로 등록 (타사코드 ↔ 우리 수집키)
  2. 우리 스크립트로 -Full 수집(이미 superset) → 타사 항목만 선택
  3. 타사 점수/등급 룰(1~5점, A 95%+ 등) 적용 → 리포트3 출력 = 타사 점검결과
  4. 매핑에서 빠진 타사 고유항목만 자율에이전트로 추가

정직한 한계

  • SRV 번호는 대표 매핑 — 일부는 정확, 일부는 점검영역 기준. 47/42 전 항목 1:1 확정은 항목별 기준문구 대조 필요.
  • 타사 점수가중치·등급기준은 데이터가 아니라 룰셋으로 별도 반영.
  • OS 버전(Solaris 9, AIX 5.x, HP-UX 11.x)은 대상별 실측 검증 권장.
